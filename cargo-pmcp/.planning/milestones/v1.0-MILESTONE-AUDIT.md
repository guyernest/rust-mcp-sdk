### E2E Flows (2/2 complete)

**Flow 1: `cargo pmcp loadtest run <URL>`**
```
main.rs → Commands::Loadtest → execute_run()
  → discover_config() / LoadTestConfig::load()     [Phase 1 → Phase 3]
  → apply_overrides()                                [CONF-02]
  → LoadTestEngine::new(config, url)                 [Phase 1 → Phase 2]
  → engine.run()                                     [Phase 2]
    → config.validate()                              [CONF-01]
    → spawns N × vu_loop()                           [LOAD-01]
      → McpClient::initialize()                      [MCP-01]
      → execute_step() → send RequestSample          [METR-01, MCP-03]
    → metrics_aggregator()                           [METR-02]
    → display_loop()                                 [METR-03]
  → render_summary()                                 [METR-04: Phase 2 → Phase 3]
  → LoadTestReport::from_result() + write_report()   [METR-05: Phase 2 → Phase 3]
```
**Status:** COMPLETE

```

```
