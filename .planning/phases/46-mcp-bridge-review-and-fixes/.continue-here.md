<context>
Key learning: MCP Apps widgets MUST use the full ext-apps SDK. Missing protocol handlers (ui/teardown, ui/toolInput, ui/toolCancelled) causes host to kill the entire MCP connection after first tool result.

For ChatGPT: server needs .with_host_layer(HostType::ChatGpt) which adds openai/* keys at build time. These propagate to resources/read via uri_to_tool_meta index and to tools/call via with_widget_enrichment().
