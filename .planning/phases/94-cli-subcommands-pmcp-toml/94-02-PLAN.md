## Trust Boundaries

|----------|-------------|
| CLI args → process | a user-supplied `.xlsx` path and `--format` string cross into the ingest + render path |

| handler error → shell exit code | a gate block (Plan 03) must reach the shell as a DISTINCT code, not be collapsed by anyhow |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
