## Customer-id → row addressing

| Option | Description | Selected |
|--------|-------------|----------|
| Simple row convention | customer_id maps to a known row; tool templates the Excel range address from the id | ✓ |
| Realistic Excel table lookup | Model as an Excel Table, look up row by id key (closer to real Graph; more surface) | |

**User's choice:** Simple row convention
**Notes:** Obviously a demo, smallest fixture, stays within the single range-read op. `get_customer_orders` reads a known contiguous orders block for the id (exact block addressing finalized in planning). No server-side filtering — richer queries are Code Mode.
