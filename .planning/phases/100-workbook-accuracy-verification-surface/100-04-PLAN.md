<objective>
Item 3 (WBVER-03): a new `verify_accuracy` meta-tool (the 6th served tool) that re-runs the executor at the workbook's reference inputs (the manifest tier defaults — VERIFIED the oracle was computed there) and returns a per-output reconciliation report vs `Tool.oracle` within `TOL` (0.01). This makes the compile-time penny-reconcile runtime-inspectable, stateless and reader-free.

Core: a new pure `reconcile.rs` in `pmcp-workbook-runtime` composing `executor::run`, a NEW pure runtime helper `seed_reference_inputs(manifest)`, and the `run.computed.get(&entry.seed_coord)` projection. Surface: a `VerifyAccuracyHandler` mirroring `ExplainHandler`, with an optional tool-name filter.

GROUNDED ARCHITECTURAL NOTE (corrects 100-RESEARCH.md lines 185/199): the toolkit's `seed_tier_defaults` is a PRIVATE fn in the TOOLKIT (`crates/pmcp-server-toolkit/src/workbook/input.rs:123`). It is NOT in the runtime crate and is NOT callable from `pmcp-workbook-runtime` — and the runtime crate MUST NOT depend on the toolkit (layering + reader-free purity). `reconcile.rs` therefore CANNOT "compose seed_tier_defaults". Instead, the runtime ALREADY carries the tier defaults natively: `crates/pmcp-workbook-runtime/src/manifest_model.rs` exposes `CellRole.tier: Option<InputTier>` where each `InputTier` variant (`Variable { default: CellValue }`, `BoundedVariable { default, .. }`) carries a typed runtime-native `CellValue` default. This plan adds a SMALL PURE runtime helper `seed_reference_inputs(manifest) -> BTreeMap<String, CellValue>` that iterates `manifest.cells`, filters `Role::Input`, and reads each cell's `InputTier` default as a `CellValue` (NOT serde_json `Value`). This mirrors the toolkit logic at the manifest-tier level WITHOUT a cross-crate dependency and WITHOUT duplicating the toolkit's dtype/enum validation (reconcile needs only the reference values, not input validation).

Purpose: Give the BA/pmcp.run a queryable attestation that the engine matches Excel's authored values at the reference inputs — transparency rather than blind trust.
Output: `seed_reference_inputs` + `reconcile_reference` + `ReconcileReport` (with the D-01 `cell: Option<String>`) + the registered 6th meta-tool, honestly framed.
</objective>
