<placeholder_recognition_rule>
<!-- REVIEWS H7 — strict colon-precedence rule for entering Placeholder state.
While in Normal state, when encountering ':':
  - PEEK the next char (do NOT consume yet).
  - If peek is `[A-Za-z_]`: consume the `:` (do not emit), switch to Placeholder state, push subsequent identifier chars to pending_name.
  - Otherwise (peek is `:`, `=`, `'`, digit, whitespace, EOF, anything else): emit ':' verbatim, stay in Normal. The next iteration handles the peeked char from scratch.

This eliminates:
  - Postgres `::text` mis-translation (`::` → emit `:`, emit `:`, then `text` is plain identifier — NOT a placeholder name)
  - MySQL `:=` session-var (`:=` → emit `:`, emit `=`)
  - Malformed `:1bad` (`:` not followed by alpha/_ → emit `:` verbatim, then `1bad` is identifier text)

NO empty-name flush path exists in the new design — entering Placeholder state already requires a valid identifier-start char, so pending_name is NEVER empty.
-->
</placeholder_recognition_rule>

    - `translate_placeholders("", Dialect::Postgres)` → `TranslatedSql { sql: "", ordered_params: [] }`

    - `("SELECT :id FROM t", Dialect::Postgres)` → `{ "SELECT $1 FROM t", ["id"] }`
    - `("SELECT :id FROM t", Dialect::MySql)` → `{ "SELECT ? FROM t", ["id"] }`
    - `("SELECT :id FROM t", Dialect::Athena)` → `{ "SELECT ? FROM t", ["id"] }`

    Replace the stub `translate_placeholders` body in `src/sql/translate.rs` with the `SqlWalker` state machine per Pattern G (split-helpers, each ≤cog 25):
    
    1. **Define `State` enum** privately in the module:
       `Normal`, `StringLiteral(char)`, `LineComment`, `BlockComment(usize /* depth */)`, `Placeholder`, `CastTypeName` (transitional after `::` — REVIEWS H7).
    
    2. **`SqlWalker<'a>` struct** with fields: `chars: std::iter::Peekable<std::str::Chars<'a>>`, `state: State`, `out: String`, `order: Vec<String>`, `pg_index: usize`, `dialect: Dialect`, `pending_name: String` (current placeholder being read).
    
    3. **`fn new(sql: &'a str, dialect: Dialect) -> Self`** — initializes all fields.
    
    4. **`fn run(&mut self)`** — outer dispatch loop; while there's a char, dispatch to handler based on `self.state`. Cog target: ~10. On EOF, if `state == Placeholder` and `pending_name` non-empty, call `self.emit_placeholder_from_pending()` (emit translated form).
    
    5. **`fn handle_normal(&mut self, c: char)`** — handles per REVIEWS H7 rule:
       - `'` or `"` → switch to `StringLiteral(c)` + emit verbatim
       - `-` peek-next `-` → switch to `LineComment` + emit verbatim
       - `/` peek-next `*` → switch to `BlockComment(1)` + emit verbatim
       - `:` peek-next:
         - `:` (double colon — Postgres cast prefix): emit both colons verbatim, consume the second `:`, switch state to `CastTypeName` (swallows the immediately-following identifier as a type name).
         - `[A-Za-z_]`: switch to `Placeholder` + clear `pending_name` (do NOT emit the `:`).
         - anything else (digit, `=`, whitespace, EOF, etc.): emit `:` verbatim, stay Normal.
       - otherwise: emit verbatim.
       Cog target: ~15. Split if needed: `fn dispatch_colon(&mut self) -> ()` helper.
    
    6. **`fn handle_placeholder(&mut self, c: char)`** — if `c` is `[A-Za-z0-9_]` → push to `pending_name`; else → emit translated form via `emit_placeholder_from_pending()` THEN re-dispatch `c` via `handle_normal`. (Because we now ONLY enter Placeholder when the next char is alpha/_, `pending_name` is guaranteed non-empty when this terminates — no empty-name flush needed.) Cog ~6.
    
    7. **`fn handle_cast_type_name(&mut self, c: char)`** — REVIEWS H7. If `c` is `[A-Za-z0-9_]` → emit verbatim, stay in CastTypeName. Else → emit verbatim, switch back to Normal. This consumes the entire type identifier following `::` so no `:` lookups happen mid-identifier. Cog ~4.
    
    8. **`fn emit_placeholder_from_pending(&mut self)`** — emit translated form per dialect (was `flush_placeholder`/`emit_placeholder`):
       - `Postgres`: `self.pg_index += 1; write!(self.out, "${}", self.pg_index).unwrap();`
       - `MySql | Athena`: `self.out.push('?');`
       - `Sqlite`: `write!(self.out, ":{}", self.pending_name).unwrap();` (identity)
       Then push `pending_name.clone()` into `self.order`. Clear `pending_name`. Reset state to Normal. Cog ~6.
    
    9. **`fn handle_string(&mut self, c: char, q: char)`** — emit `c` verbatim. If `c == q`: peek next char; if next is also `q`, emit verbatim + consume (doubled-quote escape `''`); else switch back to `Normal`. Cog ~6.
    
    10. **`fn handle_line_comment(&mut self, c: char)`** — emit `c` verbatim. If `c == '\n'`: switch to `Normal`. Cog ~3.
    
    11. **`fn handle_block_comment(&mut self, c: char, depth: usize)`** — emit `c` verbatim. Track nesting: `*` then `/` → depth-1 (Normal at depth 0); `/` then `*` → depth+1. Cog ~5.
    
    12. **`fn into_translated(self) -> TranslatedSql`** — returns `TranslatedSql { sql: self.out, ordered_params: self.order }`. Cog ~1.
