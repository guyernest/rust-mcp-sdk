<objective>
Implement secret resolution as a pre-deploy step in the cargo-pmcp deployment pipeline, with dotenvy-based .env parsing and target-specific secret handling.

Purpose: Secrets declared in server config (SecretRequirement) are resolved from local env vars and .env files before deployment. AWS Lambda gets them injected as transient CDK process env vars (never persisted to deploy.toml per D-05/D-06). pmcp.run gets diagnostic guidance only.
Output: resolve.rs module with pure resolution function, deploy pipeline wired to resolve secrets before target dispatch, DeployExecutor passing extra_env as transient CDK process env vars.
</objective>
