## Gaps Summary

Two gaps block full goal achievement:

**Gap 1 — `examples/27-course-server-minimal` compilation failure:**
This is a standalone example that is a workspace member (`exclude`d from publish but has its own `Cargo.toml` depending on `pmcp` as a path dep). When Plan 03 migrated consumers, it missed this example. The file at `examples/27-course-server-minimal/src/main.rs` uses:
- `PromptMessage { role: Role::User, content: MessageContent::Text { ... } }` (E0639 non-exhaustive + old MessageContent alias)
- `ServerCapabilities { tools: Some(...), ..Default::default() }` (E0639 non-exhaustive)

**Gap 2 — Clippy quality gate failure:**
8 clippy errors prevent `make quality-gate` from passing. Two unused `Role` imports were directly introduced by the Plan 03 migration (the migration removed `PromptMessage { role: Role::User, ... }` struct literals that were the only users of `Role`). The remaining 6 errors (needless borrows in core.rs, lifetime elision in version.rs, large_enum_variant in protocol/mod.rs) may be pre-existing — they surfaced because the compilation now reaches those files — but they must be fixed for the quality gate regardless of origin.

Both gaps are straightforward to fix: 3 file edits for the example, and targeted clippy fixes for the 5 affected source locations.
