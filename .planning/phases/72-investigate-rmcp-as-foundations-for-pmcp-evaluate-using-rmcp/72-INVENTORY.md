## Overlap Legend

| Rating | Meaning |
|--------|---------|
| EXACT | rmcp type/module matches pmcp 1:1 for spec surface; pmcp adds no behavior beyond it |
| Partial | rmcp covers most of pmcp's surface; pmcp has identifiable extra behavior or accessors |
| pmcp-superset | rmcp has a base but pmcp ships a materially richer layer (e.g., TypedTool on top of Tool) |
| pmcp-exclusive | rmcp 1.5.0 has no equivalent (verified against docs.rs) |
| UNVERIFIED | Needs direct source check in Plan 02 spike or via `cargo public-api` on an rmcp test crate |

## Serde-Compat Legend

| Rating | Meaning |
|--------|---------|
| EXACT | serde_json::to_value + from_value roundtrips bit-identically between pmcp and rmcp types |
| compatible-via-adapter | roundtrip holds with a trivial `From`/`Into` adapter (≤20 LOC per type) |
| INCOMPATIBLE | serde shape differs (field names, tagging, enum representation) — would force breaking change |
| N/A | not a serde-derived type (e.g., transport internals, handler traits) |
