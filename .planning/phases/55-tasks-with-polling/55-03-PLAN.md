<objective>
Wire TaskStore into the server builder and request dispatch, with proper capability negotiation.

Purpose: Connect the TaskStore trait from Plan 02 to the server's request handling pipeline. When a TaskStore is registered, the server advertises tasks capability via `ServerCapabilities.tasks` (the standard field, not just experimental) and dispatches tasks/get, tasks/list, tasks/cancel through the store. The existing TaskRouter trait remains for backward compat with pmcp-tasks but TaskStore is the new primary path.

Output: Updated builder, core dispatch, and re-exports for a complete tasks-with-polling workflow.
</objective>

       ```rust
       /// Register a task store for MCP Tasks with polling.
       ///
       /// When a task store is registered, the server:
       /// - Advertises `ServerCapabilities.tasks` with list and cancel support
       /// - Handles `tasks/get`, `tasks/list`, `tasks/cancel` requests via the store
       /// - Resolves task owner from auth context (OAuth subject, client ID, or session ID)
       ///
       /// # Examples
       ///
       /// ```rust,ignore
       /// use pmcp::server::task_store::InMemoryTaskStore;
       /// use std::sync::Arc;
       ///
       /// let store = Arc::new(InMemoryTaskStore::new());
       /// let server = Server::builder()
       ///     .name("my-server")
       ///     .version("1.0.0")
       ///     .task_store(store)
       ///     .build()?;
       /// ```
       #[cfg(not(target_arch = "wasm32"))]
       pub fn task_store(mut self, store: Arc<dyn crate::server::task_store::TaskStore>) -> Self {
           // Set ServerCapabilities.tasks (standard, not experimental)
           self.capabilities.tasks = Some(crate::types::capabilities::ServerTasksCapability {
               list: Some(serde_json::json!({})),
               cancel: Some(serde_json::json!({})),
               requests: Some(crate::types::capabilities::ServerTasksRequestCapability {
                   tools: Some(crate::types::capabilities::ServerTasksToolsCapability {
                       call: Some(serde_json::json!({})),
                   }),
               }),
           });

           self.task_store = Some(store);
           self
       }
       ```

    4. **Pass task_store to ServerCore::new() in `build()`:** Add the task_store parameter to the `ServerCore::new()` call. This requires ServerCore to accept an additional parameter.

           let store = Arc::new(crate::server::task_store::InMemoryTaskStore::new());
           let server = ServerCoreBuilder::new()
               .name("test")
               .version("1.0.0")
               .task_store(store)
               .build()
               .unwrap();
           let caps = server.capabilities();
