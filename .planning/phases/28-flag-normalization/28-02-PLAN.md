In `run.rs`: Remove the `detailed: bool` parameter from `pub fn execute(...)`. Inside the function, use `global_flags.verbose` wherever `detailed` was used. The function already receives `global_flags: &GlobalFlags`.

In `download.rs`: Update the handler if it receives the format as `Option<String>` -- it should now receive a `String` (since FormatValue always has a value). Or pass `Some(format.to_string())` if the handler expects `Option<String>`.
