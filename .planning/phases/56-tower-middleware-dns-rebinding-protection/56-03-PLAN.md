<objective>
Close three gaps from code review in the StreamableHttpServer path:

1. **No Tower layers in start()** — Users of `run_streamable_http()` get zero DNS rebinding protection. Fix: apply the same `DnsRebindingLayer + SecurityHeadersLayer + CorsLayer` stack that `pmcp::axum::router()` already uses.

2. **Dual CORS implementation** — `axum_router.rs` uses `CorsLayer`, `streamable_http_server.rs` uses hand-rolled `add_cors_headers()`. With Tower layers in `start()`, the hand-rolled function produces duplicate headers. Fix: delete `add_cors_headers()` entirely, delete `handle_options` (CorsLayer handles preflight), and remove all call sites.

3. **Per-request AllowedOrigins clone** — `resolve_allowed_origins()` clones `Vec<String> + HashSet<String>` on every request (5 call sites). Fix: resolve once at `ServerState` construction, store as a field.

Purpose: Every SDK user — whether using `run_streamable_http()` or `pmcp::axum::router()` — gets DNS rebinding protection by default. Single CORS implementation eliminates header divergence risk.

Output: Modified `streamable_http_server.rs` and `axum_router.rs` with unified Tower layer stack and no hand-rolled CORS.
</objective>

```rust
pub async fn start(self) -> Result<(SocketAddr, tokio::task::JoinHandle<()>)> {
    let allowed = self.state.allowed_origins.clone();

    let cors = CorsLayer::new()
        .allow_origin(allowed.to_cors_allow_origin())
        .allow_methods([Method::GET, Method::POST, Method::DELETE, Method::OPTIONS])
        .allow_headers([
            "content-type".parse().expect("valid header name"),
            "accept".parse().expect("valid header name"),
            "mcp-session-id".parse().expect("valid header name"),
            "mcp-protocol-version".parse().expect("valid header name"),
            "last-event-id".parse().expect("valid header name"),
        ])
        .expose_headers([
            "mcp-session-id".parse().expect("valid header name"),
            "mcp-protocol-version".parse().expect("valid header name"),
        ])
        .max_age(Duration::from_secs(86400));

    let app = build_mcp_router(self.state)
        .layer(SecurityHeadersLayer::default())
        .layer(DnsRebindingLayer::new(allowed))
        .layer(cors);

    let listener = tokio::net::TcpListener::bind(self.addr).await?;
    let local_addr = listener.local_addr()?;
    let server_task = tokio::spawn(async move {
        axum::serve(listener, app).await.unwrap();
    });

    Ok((local_addr, server_task))
}
```
