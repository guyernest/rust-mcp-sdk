1. **Structural config diff** — enriched `london-tube.toml` now contains `[[resources]]`, `[[prompts]]`, and `cost_hint` annotations: the same block KINDS as `reference-config.toml` (verify with `grep -c '\[\[resources\]\]\|\[\[prompts\]\]'` on both, accounting for SQL/HTTP content difference).
