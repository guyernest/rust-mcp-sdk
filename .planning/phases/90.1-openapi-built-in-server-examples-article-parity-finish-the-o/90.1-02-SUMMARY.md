## Decisions Made

- **Two resources in the walkthrough, not three.** PATTERNS Pattern B lists three candidate TfL resources; the chapter walkthrough defines two (`docs://london-tube/schema`, `code-mode://learnings`) and the prompt's `include_resources` lists exactly those two. This keeps the in-chapter example itself URI-integrity-correct (Pitfall 2) — a reader who pastes it verbatim gets a prompt whose every `include_resources` URI resolves to a defined resource, rather than a partial excerpt that would warn-skip a missing resource.
- **See-also as an absolute repo URL.** The target (`reference-config.toml`) is a source fixture outside the mdbook `src/` tree; a relative `../` link would not resolve in the rendered HTML, so both chapters use the `github.com/paiml/rust-mcp-sdk/.../reference-config.toml` reference-link form.
- **Course theme assets excluded from commits.** `pmcp-course/src/theme/exercises.{css,js}` are rewritten by the `mdbook-exercises` preprocessor on every `mdbook build` (they were already dirty in the worktree before this plan). They are build-tool output unrelated to this plan's task and were deliberately left unstaged per the executor scope boundary.
