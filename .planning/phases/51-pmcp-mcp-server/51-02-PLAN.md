Purpose: These tools let any AI agent or user test MCP servers remotely via the PMCP MCP server.
Output: Three ToolHandler implementations ready to be registered in the server builder.

IMPORTANT: Use `pmcp::Error::Internal(msg)` for error conversion since mcp-tester returns `anyhow::Error` which can't be directly converted to `pmcp::Error`.

Return the full TestReport as the response -- it already implements Serialize.
