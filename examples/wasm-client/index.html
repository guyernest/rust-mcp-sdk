MCP WASM Client Example

Rust MCP SDK - Browser-based client supporting HTTP and WebSocket transports

Connection Setup

Available Example Servers

To test the client:

  1. Run one of the example servers below in a new terminal
  2. Click "Use" to select the server URL
  3. Click "Connect" to establish connection
  4. Once connected, test the available tools
http://localhost:8081 Stateless HTTP - Run: cargo run --example 23_streamable_http_server_stateless --features streamable-http
http://localhost:3000 Stateful HTTP - Run: cargo run --example 22_streamable_http_server_stateful --features streamable-http
ws://localhost:8080 WebSocket - Run: cargo run --example 13_websocket_transport --features websocket

Custom Server: You can also connect to your own MCP server by entering its URL below. Make sure your server has CORS headers configured for browser access.
